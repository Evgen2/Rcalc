# Rcalc
Calculating the average thermal resistance of a house based on outdoor and indoor temperature data from Home Assistant and monthly gas or electricity consumption

Расчет среднего теплового сопротивления дома на основе данных о температуре наружного и внутреннего воздуха из Home Assistant и ежемесячного потребления газа или электроэнергии.

1. Запустите Rcalc в командной строке Linux/Windows. Будет создан файл конфигурации rcalc.cfg.

2. Постройте в Home Assitant график температур в помещении и на улице за месяц. Экспортируйте данные из HA в файл, например history.csv

3. Отредактируйте rcalc.cfg любым текстовым редактором.

dataFile=history.csv // путь к csv файлу

use_gas=1            // 1= используется газ 0=используется электричество

GasSHC=33500000      // Удельная теплота сгорания природного/сжиженного газа

Gas_usedV=100        // Объем потребленного газа в м3 по счетчику

Electricity_used=1000// объем потребления электроэнергии, кВт*час, если используем электричество для отопления

HomeS=100            // площадь дома в кв.м

HomeCS=320	     // площадь ограждающих конструкций дома, (стены+пол+потолок) м2


4. Снова запустите Rcalc.

Полученное среднее сопротивление теплопередаче ограждающей поверхности R в м2*С/Вт можно использовать для сравнения с 
сопротивлением теплопередаче различных материалов, например на сайте 
[Теплотехнический калькулятор ограждающих конструкций](https://www.smartcalc.ru/thermocalc)

Кроме того, полученное значение R можно использовать для оценки эффективности работ по утеплению дома или изменению его энергоэффективности при выпадении снега, при деградации тепловой защиты и т.п.
